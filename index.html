Robot Arena

A Unity-based battle sandbox where you program robot behavior with Scratch-like visual blocks, tune loadouts, and watch your strategy fight in a deterministic physics arena.

Unity 2022.3 LTS Visual Block Editor Deterministic Physics

Core Systems

Robot Arena is built on four major pillars that work together to create the battle experience.

🧩

Visual Block Programming

Drag-and-drop Scratch-style blocks from a palette into a workspace. Blocks snap together vertically and nest inside control structures. Supports variables, custom voids, event triggers, and full JSON import/export — you can even hand-author strategies as JSON files.

🔧

Accessory Loadout System

Equip wheels, weapons, armor, sensors, and special modules. Each accessory consumes power, modifies stats (speed, damage, sensor range, etc.), can unlock exclusive blocks, and may grant special abilities like Bouncer (radial knockback) and Freezer (enemy slowdown).

⚙️

Deterministic Simulation

Matches run on a fixed 60 Hz tick clock with a custom physics world. This means results are reproducible — same program, same outcome every time. Includes projectile ricochet, knockback, health/damage, and robot-falling-off-map detection.

📡

Sensor & Event System

Robots can sense front clearance, map edges, enemy distance/direction/health/speed, bullet detection, and wall/enemy contact. Event blocks scan conditions every 0.1s and can interrupt normal execution via priority queues — enabling reactive behaviors.

Block Categories

Blocks are organized into categories in the palette. Select a category on the left to see its blocks. 44 block types are available, covering movement, sensing, logic, math, events, variables, and special accessories.

ACTION CONTROL SENSING OPERATORS EVENTS SPECIAL

What each category covers

ACTION

Move, Rotate, Wait, Fire, AimAtEnemy, PointTowards, and WaitUntil — these directly control your robotʼs body and cannon.

CONTROL

Repeat (forever), RepeatTimes, If, IfElse, and RepeatUntil — these structure program flow with loops and branching.

SENSING

FrontClear, FrontIsMapEdge, EnemyInView, ContactWithEnemy, ContactWithWall, DetectBullet — these query the environment.

OPERATORS

Math (Add, Subtract, Multiply, Divide), comparisons (>, <, =), logic (And, Or, Not), and value reporters like Distance, Health, Direction, and Speed.

VARIABLES & VOIDS

Create named variables with Set, Change, Show, and Hide; reporter blocks read their values. Custom Voids let you save reusable subroutines.

SPECIAL

ActivateBouncer and ActivateFreezer — execute accessory-gated abilities that deal radial knockback or temporarily freeze the enemy.

Accessory System

Accessories change how your robot performs. Each one costs power — you have a limited budget, so choose wisely. The ACCESSORIES tab lets you equip and preview parts.

🛞 Wheels

Affect move and rotate speed via duration multipliers.

🔫 Weapons

Modify muzzle velocity, fire rate, and damage multipliers.

🛡️ Armor

Grant bonus health and sometimes sensor range boosts.

📡 Sensors

Extend detection range for enemy visibility and distance queries.

⚡ Specials

Unlock unique abilities: Bouncer (radial force push, configurable radius and target layer) and Freezer (temporary enemy speed debuff with cooldown).

🔓 Block Unlocks

Some accessories unlock blocks that are otherwise hidden in the palette — you must equip the right part to access certain abilities.

JSON Strategy Format

A program is a flat JSON graph. Key rules:

  • startBlockId points to the Start block
  • nextId chains blocks vertically; childSlots attach control bodies and conditions
  • Block types are stored as numeric enum values (see strategy guide for the mapping)
  • Variables and custom voids are defined at the root level
  • Accessory loadout can be included when exporting
Strategy tips: Use EnemyInView before firing — walls block line of sight. Use FrontClear + FrontIsMapEdge before moving. Variables are great for state-machine phases (routing → search → attack → recover). Event blocks with priorities let you react instantly without polling in the main loop.

Getting Started

1. Open in Unity

Open the project folder in Unity 2022.3.62f3c1 and load Assets/Game.unity.

2. Press Play

Hit Play in the editor. The programming UI and 3D arena appear side-by-side.

3. Build a Program

Drag blocks from the left palette into the workspace. Snap them together under Robot Start.

4. Run the Match

Press > Run. Watch your robot execute the program and fight the enemy in real time.
